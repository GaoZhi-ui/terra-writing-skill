## 核心三角

```
            沈默
           /    \
        阿枣    古籍修复师
          |        |
       卫大姐   陈晖洁
```

**沈默 → 阿枣**：互相守护。沈默给了她一个"迟早要回来"的杯子，她替他收着刻符石头。
**沈默 → 古籍修复师**：信息提供者。不等同于信任——沈默还不知道他全部的目的。
**阿枣 → 卫大姐**：姐妹。三年后重逢，已拿到三孔数据册。
**古籍修复师 → 陈晖洁**：通过她联系沈默。陈晖洁打了招呼让沈默顺利过检查站。

## MC-0729圈子

```
                   沈默
                 /  |  \
              老张 老方 刘婶
               |    |    |
             阿远  大壮 阿枣
```

- **老张**：给沈默第一把工具的人，师徒关系。送了水袋，提了兄弟的事。
- **老方**：最早的客人，送搪瓷缸+留言"到了写信"。
- **刘婶**：用食物往来代替语言。
- **阿远**：萨弗拉采集工，送画，说"草绳三个结的意思是还会回来"。
- **大壮**：鲁珀装卸工，送扳手。

## 永宁圈子

```
         卫大姐
           |
         阿枣
           |
         沈默 —— 修书老人 —— 书铺菲林
           |
         左林
```

- **修书老人**：符号信息的第一个外部来源。
- **书铺菲林**：指路者——指引沈默去找修书老人。
- **左林**：龙门线司机，提供灰烬平原的情报。
- **卫大姐**：阿枣的姐姐，保存了三孔数据。

## 龙门圈子

```
        古籍修复师
         /      \
      沈默    陈晖洁
       |
     阿枣
```

- **古籍修复师**：符号坐标的持有者。阿戈尔人。等沈默两个月。
- **陈晖洁**：龙门近卫局特别督察。舅父魏彦吾。通过古籍修复师联系沈默。
