# 写作风格速查表

> v3 | 动笔前扫一眼，防止跑偏

---

## 字数·结构

| 项目 | 标准 |
|------|------|
| 正文 | 2500汉字/章，±300 |
| 日记 | 80-120字，三样：一个事实+一个观察+一个画面 |
| 日记不记 | 感受·总结·哲学追问 |
| 场景切换 | `—` 分隔 |

## 留白（最重要）

- ✅ 保留：场景过渡·多一轮对话·身体细节
- ❌ 不加：帮读者总结的段落·哲学追问·往主题方向推的对话
- 金线：删掉这句，场景的呼吸还在不在？

## 对话

| 规则 | 说明 |
|------|------|
| 辨识度 | 每个角色说话方式不同——阿枣短促直接，老张慢而沉，刘婶绕弯子 |
| 萨卡兹式 | 碎片化短句，沉默比句子长 |
| 不做主题推手 | 让他们说自己的事 |
| 标签多样性 | 用动作伴随对话，不用"他说/她说"堆砌 |

## 信息释放

- **地震级信息**用最短的句子："耳朵是圆的。在头两侧。"
- 沈默不解释动机，赫德雷不解释石头
- 读者能看见的，就不帮读者说出来

## 场景构建

- 每个场景：**环境锚点 + 动作/对话 + 情绪节拍 + 信息推进**
- 每800字左右出现一次钩子

## 五感检查

每场景至少2种：视/听/嗅/味/触

## 比喻禁忌

❌ 互联网·手机·外卖·微信·美团——泰拉世界没有这些

## 不要做的事

- 不让沈默"什么都懂"
- 不让配角变工具人——每个角色有自己的动机
- 不用"他心想/他觉得"偷懒
- 不每章都"发现一件新事"——有些章该写旧事发酵

## 字数不达标怎么办

不够2500 → 加：过渡场景·多一轮对话·人物身体细节
不靠：解释性段落·帮读者总结

## 新增写作技法参考

### 冰山温度计
写完删掉最像"替读者总结"的那句话——删完场景的呼吸还在不在？（改编自海明威冰山理论）

### 微观张力检查
本章有多少"表面说A实际在说B"的对话？少于3处则潜台词不足。

### 白房间检查
每段对话开始后三句内应有环境锚点——不让对话发生在真空里。

### 场景结构
每章想清楚是Scene（推进）还是Sequel（消化），连续两章同类型要调节奏。

### 主动性追踪
章末锚点文件加一行：沈默主动性（1-10）。1=完全被动（事找人），10=主动决策找人。核心弧线是从被动到主动，每章问自己：这章是推了还是拉了？
