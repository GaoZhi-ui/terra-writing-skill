# 因果链追踪

> 故事真正的骨架。伏笔表是"有什么线索"，因果链是"事情为什么发生"。
> 每条链 = 因为A → 所以B → 所以C → 所以D。节点之间必须有因果关系。

---

## 主链一：符号之谜（完整链·至Ch55）

```
牧人从玻利瓦尔来，在拾遗画了符号（Ch37）
  → 因为沈默记住了这个符号
  → 所以补给站老板娘送来刻符石头时他立刻认出（Ch39）
  → 所以陈晖洁的信让他决定去龙门（Ch39）
  → 所以他在永宁书铺注意到带同符号的空白书（Ch42）
  → 所以书铺老板指路→他去找修书老头（Ch43）
  → 所以修书老头确认符号是"确认"标记，地下有东西（Ch43）
  → 所以沈默去永宁工地实地勘查（Ch45）
  → 所以发现铁板焊死+地底声音→决定去龙门（Ch45）
  → 搭左林车→到龙门→见古籍修复师（Ch46-48）
  → 古籍修复师说符号是坐标网络（Ch48）
  → 陈晖洁提供车（Ch49-50）
  → 古籍修复师在地图上画叉——六个标记围成一扇门（Ch50）
  → 车到灰烬平原→金属杆路标→找到埋地门（Ch52）
  → 石头没反应→阿枣发现门在下面→盖板下通道（Ch52-53）
  → 石头开门→冷蓝色大厅→圆形图案（Ch53）
  → 石头放入凹陷→图案激活→看到七个设施的三维图（Ch54）
  → 回到龙门→古籍修复师翻出路线笔记→第一个坐标：北边盐碱地（Ch55）
```

**当前状态**：链到Ch55。
**下一节点**：往北去盐碱地→废弃城市骨架→第二设施。

## 主链二：阿枣的身世与秘密

```
阿枣在永宁东区工地发现三孔异常数据（三年前）
  → 因为数据被删、编号重排
  → 所以她离开永宁去了MC-0729（Ch33）
  → 所以王工程师找上门来确认她还活着（Ch35）
  → 所以老方每月托人带话给卫大姐"她还活着"（持续）
  → 所以阿枣回永宁后拿到原始数据册（Ch44）
  → 所以卫大姐说"钻头声音变了，不是石头，是空的声音"（Ch44）
  → 所以沈默把三孔数据纳入符号地图的第四条线（Ch45）
```

**当前状态**：已汇入主链一。

## 主链三：古籍修复师—路线笔记

```
古籍修复师年轻时在炎国边境商队当记录员
  → 记了几十年的路线笔记
  → 发现所有旧路从不同方向汇聚到同一区域
  → （此时他还没意识到这些路和符号有关）
  → 一个从玻利瓦尔走到龙门的人告诉他：门在地下
  → 古籍修复师在龙门等了两个月
  → 陈晖洁的人把信送到MC-0729（Ch39）
  → 沈默来了（Ch47-48）
  → 灰烬平原地下大厅证实了笔记的指向（Ch53-54）
  → 古籍修复师翻出笔记→第一个坐标：盐碱地·废弃骨架下方（Ch55）
```

**当前状态**：笔记将在后续旅程中持续提供路线线索。
